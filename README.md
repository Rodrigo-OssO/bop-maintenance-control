# Bop Maintenance Control

## Project Overview
Bop Maintenance Control is a tool designed to facilitate the management and tracking of maintenance activities for various assets. It provides a user-friendly interface for logging maintenance work, scheduling upcoming tasks, and generating reports.

## Features
- **Asset Management**: Keep track of all assets requiring maintenance.
- **Schedule Maintenance**: Schedule and assign maintenance tasks.
- **Reporting**: Generate detailed reports on maintenance activities.
- **User Management**: Manage users and assign roles for access control.

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Rodrigo-OssO/bop-maintenance-control.git
   ```
2. **Navigate to the project folder**:
   ```bash
   cd bop-maintenance-control
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Run the application**:
   ```bash
   npm start
   ```

5. **Access the application** at `http://localhost:3000`.
